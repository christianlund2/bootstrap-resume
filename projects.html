Very Varied Quiz

The Very Varied Quiz is a Pop Culture quiz built in JavaScript, HTML and CSS. It includes 10 questions based on TV, Movies and Music. It features a score counter beneath the quiz container that updates as the user progresses. At the end of the game, the user can choose to reload the game and play again or submit their score.

Battleship

Battleship is a Python terminal game running on Heroku. The user tries to guess the position of five randomly generated ships on a 7x7 grid. The user can choose to play on "Easy" or "Difficult" mode (8 shots and 5 shots). It validates user input on the grid for valid shots and duplicates, hits and misses, and number of shots remaining. At the conclusion of the game, the user is informed that either no shots remain or all ships have been destroyed.
